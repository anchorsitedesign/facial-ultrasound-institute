# Dawn Naylor DNP Facial Ultrasound Institute — Project Playbook

## Project Purpose

This website is the digital home for the Dawn Naylor DNP Facial Ultrasound Institute.

The goal is to establish trust, authority, and educational credibility for advanced facial ultrasound training in aesthetics.

The site should feel premium, editorial, calm, clinical, and highly credible.

## Current Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- GitHub
- Vercel

## Development Workflow

1. Build locally in VS Code.
2. Preview on localhost.
3. Replace full files whenever practical.
4. Test the build.
5. Commit changes.
6. Push to GitHub.
7. Review live deployment on Vercel.
